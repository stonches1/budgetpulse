Questions Fréquentes

Mes données financières sont-elles privées ?

Oui ! Toutes vos données sont stockées localement sur votre appareil. Nous ne collectons, n'accédons ni ne transmettons jamais vos informations financières vers des serveurs. Votre budget reste totalement privé.

Comment restaurer mon achat premium ?

Ouvrez BudgetPulse, allez dans Paramètres, appuyez sur « Passer à Pro », et recherchez l'option « Restaurer les achats ». Assurez-vous d'être connecté avec le même identifiant Apple utilisé pour l'achat initial.

Puis-je exporter mes données de budget ?

Oui ! Allez dans Paramètres > Exporter les données pour sauvegarder vos transactions, catégories et historique de budget. C'est utile pour sauvegarder vos données ou les analyser dans un tableur.

Comment fonctionnent les notifications de budget ?

BudgetPulse envoie des notifications locales (générées sur votre appareil, pas depuis un serveur) pour les avertissements de limite de budget, les rappels de dépenses et les célébrations d'objectifs. Vous pouvez les personnaliser ou les désactiver dans les Paramètres.

J'ai trouvé un bug. Comment le signaler ?

Veuillez nous envoyer un email à badue043@gmail.com avec une description du problème, le modèle de votre appareil et la version iOS. Les captures d'écran sont très utiles !
